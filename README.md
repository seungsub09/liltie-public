# Liltie

음악과 마음을 매일 한 곡으로 기록하는 iOS 앱.
오늘 들은 음악, 메시지, 무드 물결로 하루를 남겨보세요.

---

## 한국어

### 주요 기능

- **하루 한 곡 기록** — 곡 검색 → 메시지 + 무드 단계 5단계 선택 → 저장. 하루 한 개의 기록으로 부담 없이.
- **무드 물결** — 차분한~거센 5단계를 시각적인 물결로 표현
- **나만의 물결을 만들어가다** — 기록된 하루는 하나의 물결이 되고, 그 물결들이 모여 나만의 흐름이 됩니다. 지난 7일 / 캘린더 뷰에서 인접한 날의 앨범 컬러가 부드러운 그라데이션으로 이어지고, 잔잔한 애니메이션으로 매일의 무드를 다시 떠올리게 해줍니다.
- **Apple Music 통합 (선택)** — 권한 허용 시 Apple Music 카탈로그 검색 + 최근 들은 곡 빠르게 선택
- **iTunes Search 폴백** — Apple Music 미연동 시에도 한국 스토어프론트 기반 풀 검색
- **30초 미리듣기** — iTunes 공식 preview 활용, 재생 중 LP 회전 + 물결 흐름 애니메이션
- **외부 음원 앱 연결** — Apple Music / Spotify / YouTube Music으로 한 번에 이동
- **인스타 스토리 공유** — 앨범 커버 + 곡 정보 + 무드 물결로 구성된 sticker, 팔레트 배경
- **iCloud 자동 동기화** — 회원가입 없이 Apple ID 기반 CloudKit 동기화
- **테마** — 밝게 / 어둡게, 앱 강제 적용

### 시스템 요구사항

- iOS 18.0 이상
- iPhone (Portrait)
- Apple Music 통합은 Apple Music 구독 또는 무료 미리듣기 사용자만 활용 가능

## English

### Highlights

- **One song a day** — Search a track, add a short note, pick one of 5 mood levels, save. One record per day, no pressure.
- **Mood waveform** — Calm to intense visualized as a 5-step waveform
- **Lookback** — Last 7 days / calendar views with cross-cell gradient + sequential animation
- **Apple Music integration (optional)** — Catalog search + recently played when authorized
- **iTunes Search fallback** — Full search via Korean storefront when Apple Music is not connected
- **30-second previews** — Official iTunes previews with rotating LP + waveform animation
- **External music app links** — One tap to Apple Music / Spotify / YouTube Music
- **Instagram Stories sharing** — Sticker with cover + track + mood waveform
- **iCloud auto-sync** — No sign-up; Apple ID-based CloudKit sync
- **Theme** — Light / Dark, forced app-wide

### Requirements

- iOS 18.0+
- iPhone (Portrait)
